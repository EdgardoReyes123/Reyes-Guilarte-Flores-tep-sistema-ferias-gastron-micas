# Reyes-Guilarte-Flores-tep-sistema-ferias-gastron-micas
Este proyecto consiste en el diseño e implementación de un sistema distribuido basado en microservicios para la gestión integral de ferias gastronómicas. Desarrollado bajo el marco de la asignatura Tópicos Especiales de Programación, el backend utiliza un stack tecnológico robusto compuesto por TypeScript, NestJS y PostgreSQL
